# Day 5: Password Generator

## Objectives
- Build a strong password generator.
- Master Python **Loops** (`for` loops with `range()`).
- Learn to manipulate **Lists** (appending, shuffling).
- Generate random inputs using `random.choice()` and `random.randint()`.

## Exercises
- **Average Height**: Calculating average without `sum()` or `len()`.
- **High Score**: Finding max value without `max()`.
- **FizzBuzz**: Classic interview question using modulo and conditionals.

## How to Run
```bash
python password_generator.py
```
