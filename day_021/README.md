# Day 21: Snake Game (Final)

## Objectives
- Complete the Snake Game from Day 20.
- Understand **Class Inheritance** (inheriting from `Turtle`).
- Implement **Slicing** in Python lists.
- Add collision detection (with food, walls, and tail).

## Features
- **Food**: Randomly appears on screen. Eating it increases score and length.
- **Scoreboard**: Tracks current score using a custom class.
- **Game Over**: Triggers when hitting the wall or own tail.

## How to Run
```bash
cd snake-game-final
python main.py
```
