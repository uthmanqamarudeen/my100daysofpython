# Day 4: Rock Paper Scissors

## Objectives
- Build a Rock Paper Scissors game against the computer.
- Master **Lists** and **Random module**.
- Use `random.choice()` for computer selection.
- Implement game logic with conditionals.

## Exercises
- **Banker Roulette**: Randomly selecting who pays the bill.
- **Treasure Map**: Nested lists and 2D indexing.

## How to Run
```bash
python rock_paper_scissors.py
```
