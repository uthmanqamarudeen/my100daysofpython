# Day 8: Caesar Cipher

## Objectives
- Create a Caesar Cipher program to encrypt and decrypt messages.
- Use `functions` with **inputs** (arguments and parameters).
- Master list indexing and handling user inputs.

## My Learnings
- **Functions with Inputs**: Understanding the difference between *parameters* (variable name) and *arguments* (actual data).
- **Positional vs Keyword Arguments**: Calling functions by position `func(a, b)` vs by name `func(b=2, a=1)`.
- **List Indexing & Modulus**: Using the `%` operator to wrap around the alphabet list preventing `IndexError` when shifting beyond 'z'.


## How to Run
```bash
python caesar_cipher/caesar_cipher.py
```
