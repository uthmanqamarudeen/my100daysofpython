# Day 22: Pong Game

## Objectives
- Build the classic **Pong Game** using Turtle Graphics.
- Apply **OOP** with multiple classes (Paddle, Ball, ScoreBoard).
- Implement ball physics (bouncing off walls and paddles).
- Create a two-player game with keyboard controls.

## How it Works
- **Left Player**: W (up), S (down)
- **Right Player**: Up Arrow, Down Arrow
- Ball bounces off top/bottom walls and paddles.
- Miss the ball? Opponent scores a point!

## How to Run
```bash
cd pong-game
python main.py
```
