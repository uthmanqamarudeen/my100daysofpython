# Day 18: Turtle Graphics

## Objectives
- Learn the **Turtle Graphics** module in Python.
- Create shapes, patterns, and art with code.
- Build a Hirst Painting (dot art) inspired by Damien Hirst.
- Use RGB colors and randomization.

## Projects
- **Shapes & Patterns**: Drawing various geometric patterns.
- **Hirst Painting**: Creating a 10x10 grid of colored dots.

## How to Run
```bash
cd hirst-painting-start
python main.py
```
