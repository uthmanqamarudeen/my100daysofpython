# Day 24: Files, Paths, and Directories

## Objectives
- Learn to work with **Files** in Python (reading and writing).
- Understand **file paths** (absolute vs relative).
- Practice **File I/O** operations (`open()`, `read()`, `write()`).
- Apply file operations to real projects.

## Projects

### 1. Snake Game (with High Score)
- Enhanced Snake Game from Day 21 to **save high scores**.
- Reads from `data.txt` on startup, writes new high score when beaten.
- **Key Skill**: Persistent data storage.

### 2. Mail Merge Project
- Automatically generates personalized letters from a template.
- Reads names from `invited_names.txt` and replaces placeholders in a letter template.
- Saves each personalized letter to the `Output` folder.
- **Key Skill**: File manipulation and string replacement.

## How to Run

**Snake Game:**
```bash
cd snake-game
python main.py
```

**Mail Merge:**
```bash
cd "Mail Merge Project Start"
python main.py
```
