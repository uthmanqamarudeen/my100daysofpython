# Day 3: Treasure Island

## Objectives
- Build a text-based adventure game (Treasure Island).
- Master **Conditional Statements** (`if`, `elif`, `else`).
- Use **Logical operators** (`and`, `or`, `not`).
- Create branching narratives with nested conditionals.

## Exercises
- **Leap Year**: Calculating leap years with complex conditions.
- **Pizza Delivery**: Price calculator with multiple conditions.
- **Love Calculator**: Fun score generator based on name inputs.

## How to Run
```bash
python my_story.py
```
